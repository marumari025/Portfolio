TypingGame

概要
使用言語 : cpp
使用ライブラリ : DXライブラリ
制作人数 : １人

説明
この作品はcppとdxライブラリを使ってエンジンを自作し、
そのエンジンを使用し作成したゲームです。

工夫
このゲームで使っているエンジンは、アップデート、FPSで分けているので、
描画と、処理を分けることで軽く動くようにしています。

ゲーム説明
真ん中に表示された単語の赤くなっている文字を入力すると
スコアが１増えるので、
時間切れになるまでにどれだけタイピングできたかを競うゲームです。
終了後、3位までのスコアを超えたときは、
自分のスコアが赤く表示されランキングに入ります。

DDD

概要
使用言語 : unity
制作人数 : 6人
制作担当 : プレイヤー

説明
この作品はunityで作成したゲームです。
学校のゲームジャムという即席チームで3日でゲームを作る
イベントで、3日間で作成しました。

工夫
敵をよける状態になったとき、
敵と重なっているならよける状態がずっと続くようにしたこと、
敵から離れた瞬間によける状態をすぐ解除するようにしました。

ゲーム説明
敵をジャンプか、よける能力で避け続けてハイスコアを目指すゲームです。
敵の近くでよけると、得点が上がるようになっています。
薄い色の壁はよける能力じゃないと通り抜けられません。

SnakeBlock

説明：テスト（SnakeBlock）
概要
使用言語 : unity
制作人数 : 4人
制作担当 : タイトル画面

説明
この作品はunityで作成したゲームです。
学校のゲームジャムという即席チームで3日でゲームを作る
イベントのクラス版のイベントで、3日間で作成しました。

自身の工夫
自作でドットのフォントのようなものを簡単に作れるものを制作し、
スタートボタンやタイトルなどで使いました
タイトルでブロックがずっと動くようにして目を引きやすいようにして、
タイトルから移動するときにアニメーションを入れて、
できるだけ楽しめるように頑張りました

ゲーム説明
プレイヤーが陣地から出ると自分の後ろにしっぽを残し、
それに自分や敵が当たったらゲームオーバーになります。
自分や敵に当たらずに自陣に帰ることができると、
尻尾で囲んだ部分が陣地になります。
これで敵を倒すというのがこのゲームです。
